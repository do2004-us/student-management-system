# Student Management System Project Summary

## Project Title

Student Management System

## Project Description

The Student Management System is a web-based application designed to help schools manage students, teachers, classes, subjects, attendance, results, fees, payments, reports, and notifications.

## Technologies Used

- HTML for page structure
- CSS for styling and responsiveness
- JavaScript for future interactivity
- PHP for server-side logic
- MySQL for database storage
- XAMPP for local development

## Main Users

- Admin
- Teacher
- Student

## Admin Features

- Login and logout
- Dashboard statistics
- Manage students
- Manage teachers
- Manage classes
- Manage subjects
- Assign subjects to classes
- Assign teachers to classes and subjects
- Manage academic terms
- Manage school fees
- Record payments
- Generate reports
- Send notifications
- Change password
- CSRF protection

## Teacher Features

- Login and logout
- View teacher profile
- View assigned classes and subjects
- Mark attendance
- Upload results
- View notifications
- Change password

## Student Features

- Login and logout
- View profile
- View attendance
- View results
- View fee status
- View notifications
- Change password
